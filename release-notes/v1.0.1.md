# v1.0.1

## Summary

Initial public-ready release of Workday Timesheet Helper.

## Included

- Clock picker added to Workday numeric timesheet widgets
- Standard-hour to Workday fractional-hour conversion menu
- Read-mode and edit-mode display handling for Workday widgets
- Green highlighting for standard converted values
- Amber highlighting for non-standard non-zero values
- Popup conversion reference table
- Updated toolbar and popup icon artwork
- Privacy policy and Chrome Web Store listing assets

## Packaging

- Publishable zip can be rebuilt with `npm run release:build`
- Current version: `1.0.1`
