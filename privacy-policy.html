Privacy Policy for Workday Timesheet Helper

Effective date: 23 March 2026

Workday Timesheet Helper is a browser extension that assists with entering hour values into supported Workday timesheet fields.

What the extension does

The extension runs on supported Workday pages and adds a small in-page picker next to timesheet hour inputs. When a user selects a value, the extension inserts the corresponding converted hour value into the page.

Information the extension accesses

The extension accesses page elements required to identify supported Workday timesheet hour fields and update those fields in the browser.

Data collection and transmission

  • The extension does not collect personal data for external storage.
  • The extension does not send timesheet values or page content to external services.
  • The extension does not use analytics, tracking, or advertising services.

Data storage

The current version of the extension does not store user-entered timesheet data outside the active browser page.

Permissions

The extension requests access to supported myworkday.com pages only so it can find Workday timesheet hour fields and attach the in-page conversion picker.

Changes to this policy

If the extension's data handling changes in a future release, this policy should be updated before that release is published.
